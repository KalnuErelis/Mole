Command modules with shared UX.

Every flow sticks to the same keybindings and a focused, minimal layout.

mo clean

Audit caches, logs, and app leftovers with a summary and dry-run mode.

  • Whitelists for protected paths
  • Category totals per cleanup step
  • Explicit deletions only
mo uninstall

Pick apps, view related files, and remove everything with a single action.

  • Top offenders ranked by size
  • Recent vs old app grouping
  • Path previews before delete
mo analyze

Zoom into directories and sort by size or age with responsive navigation.

  • Worker pool optimized for speed
  • File age badges for context
  • Quick open and delete actions
mo status

Stream live system metrics and a health score with zero setup.

  • CPU, memory, disk, power stats
  • Real-time network throughput
  • Top process snapshots
